This `ImageNet-exp` directory is adapted from the fast ImageNet training code released with PuzzleMix:
https://github.com/snu-mllab/PuzzleMix

Within this code path, the training scaffold also traces back to:

- FreeAdversarialTraining:
  https://github.com/mahyarnajibi/FreeAdversarialTraining
- PyTorch ImageNet example:
  https://github.com/pytorch/examples/tree/master/imagenet

The current repository contains the DropMix-specific modifications used for the paper experiments.
